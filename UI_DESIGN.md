## Color Scheme

### Primary Colors (WhatsApp-like)
- **Header**: #075e54 (Dark teal)
- **Message Bubble**: #dcf8c6 (Light green)
- **Background**: #e5ddd5 (Light beige with pattern)
- **Input Area**: #f0f0f0 (Light gray)
- **Icons**: #25d366 (Green accent)

### Typography
- **Font**: Roboto, Segoe UI, sans-serif
- **Message Text**: 14-16px
- **Timestamps**: 12px, light gray
- **Headers**: 24-32px, bold

## Features Shown in UI

1. **Top Bar**
   - App title
   - Search icon (magnifying glass)
   - Date picker icon (calendar)
   - Logout icon (door/exit)

2. **Message Area**
   - Date separators (Today, Yesterday, etc.)
   - Message bubbles (right-aligned, green background)
   - Timestamps below each message
   - Delete button (appears on hover)
   - Image preview in bubbles
   - Scrollable content

3. **Input Area**
   - Attachment icon for images
   - Multi-line text input
   - Send button (only enabled when content exists)
   - Image preview with remove option

4. **Dialogs**
   - Search dialog with text input
   - Date picker with calendar selector
   - Clean, centered modal design

## Responsive Breakpoints

- **Mobile**: < 768px
  - Single column layout
  - Smaller message bubbles
  - Compact icons
  - Touch-optimized buttons

- **Tablet**: 768px - 1024px
  - Medium-sized message bubbles
  - Adequate spacing
  - Touch-friendly

- **Desktop**: > 1024px
  - Maximum width container
  - Larger message bubbles
  - Hover states
  - Mouse-optimized

## Accessibility Features

- Keyboard navigation support
- Screen reader labels
- High contrast colors
- Touch target sizes (44x44px minimum)
- Focus indicators
- ARIA labels on icons

## Animation & Transitions

- Smooth scroll to new messages
- Fade in for new messages
- Slide in for dialogs
- Hover transitions (0.2s)
- Loading spinners
